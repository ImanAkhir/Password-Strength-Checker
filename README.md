# Password Strength Checker (Python)

A simple and effective **Password Strength Checker** built in Python.  
Perfect for beginners, cybersecurity learners, and portfolio projects.

---

## 1. Features
- Evaluates password based on:
  - Length  
  - Uppercase / lowercase letters  
  - Numbers  
  - Special characters  
  - Weak/common password list  
- Gives a **score (0â€“10)** and rating (**Weak / Moderate / Strong**)  
- Provides suggestions to improve weak passwords  
- Runs entirely in the **command line**

---
## 2. Usage

Run the program:
python password_checker.py

Example output:
Password Strength Checker
Enter a password to check: JWYIVE10!

Strength: Strong (Score: 10/10)
Your password looks strong!
